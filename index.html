Curso Maratona Java Virado No Jiraia

Estas são as minhas anotações (incluindo os códigos) dos meus estudos dentro do curso "Maratona Java Virado No Jiraia", da Dev Dojo Academy. O curso pode ser visto integral e gratuitamente neste link:

Este curso, criado e ministrado por William Suane, pressupõe que o estudante já conheça a Lógica de Programação. De resto, qualquer pessoa pode acompanhar o curso, que ensina Java desde o zero até muitíssimo avançado, totalizando mais de 280 vídeos.

Aos poucos, vou preenchendo as anotações efetivamente, para cada vídeo-aula em separado.

Os códigos foram escritos em um projeto Java usando a IDE IntelliJ Community Edition.

public class Welcome {
public static void main(String[] args) {
System.out.println("Olá, Mundo! Sejam todes bem-vindes ao Java!");
}
}

Depois do vídeo “00” (de boas-vindas) e do vídeo “01” (com uma descrição geral do conteúdo do curso), partimos para as aulas, começando pelo vídeo “02”, embora ele seja o terceiro na playlist, já que o primeiro é o vídeo “00”.
